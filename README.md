# Forkploy
 Forkploy is fork of Dokploy a free, self-hostable Platform as a Service (PaaS) that simplifies the deployment and management of applications and databases.

## ✨ Features

Forkploy includes multiple features to make your life easier.

- **Fucking context menus**: Right click on a service or project to interact with it instead of using bulk ops.
- **Individual container actions**: Do actions against individual containers in a compose(restart, stop blah blah)
- **Container migrations**

## 🚀 Getting Started

To get started, run the following command on a VPS with dokploy installed:

docker pull ghcr.io/irazvan2745/dokploy:VERSION

docker service update --image ghcr.io/irazvan2745/dokploy:VERSION dokploy
